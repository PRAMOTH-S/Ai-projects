# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project predicts the likelihood of heart disease using machine learning algorithms.  
It analyzes medical attributes such as age, cholesterol, blood pressure, and heart rate to assist in early diagnosis.

---

## 🚀 Features
- Machine learning-based prediction system  
- Data preprocessing and feature selection  
- Model training and evaluation  
- Simple prediction script  

---

## 🧠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn
